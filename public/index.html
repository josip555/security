  • Broken authentication is a broad term for several vulnerabilities that attackers exploit to impersonate legitimate users online. Usually, broken authentication refers to weaknesses in two areas: session management and credential management. For example attackers employ huge credential stuffing attacks to highly targeted schemes aimed at gaining access to user's username and password.
    Source: auth0.com

  • Cross-site scripting (XSS) is a type of security vulnerability that can be found in some web applications. XSS attacks enable attackers to inject client-side scripts into web pages viewed by other users. A cross-site scripting vulnerability may be used by attackers to bypass access controls such as the same-origin policy.
    Source: en.wikipedia.org

  • Cross site request forgery (CSRF) is an attack vector that tricks a web browser into executing an unwanted action in an application to which a user is logged in. CSRFs are typically conducted using malicious social engineering, such as an email or link that tricks the victim into sending a forged request to a server.
